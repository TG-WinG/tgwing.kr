# Project Introduction

**TGWinG 공식 웹사이트 😆**<br />
경희대학교 컴퓨터공학과 학술동아리 T.G.WinG 공식 웹사이트입니다. 기존의 웹사이트를 대체하여 테크 블로그 형식의 새로운 웹사이트를 제작하였습니다. <br />

**학습한 내용을 기록하고 공유해요 📚** <br />
Velog와 같이 블로그 형식으로 학습한 내용을 기록하고 공유할 수 있도록 제작하였습니다. 프로젝트 진행 경험이나 전공과 관련된 분야 외에도 본인의 관심사를 공유해보세요.

**동아리원들과의 프로젝트 경험 💻** <br />
TGWinG 동아리원들과 진행했던 프로젝트 경험을 작성할 수 있습니다(타 동아리원들도 멤버로 기입 가능). 본인의 자랑스러운 프로젝트 경험을 공유하고 정리해보세요. <br />
프로젝트 시작이 어렵다고 생각되나요? 동아리 선후배들이 진행한 프로젝트를 참고하며 최고의 프로젝트를 만들어보세요!

# Start Guide

### Requirements

최소사양으로 더 높은 버전을 사용해도 무방합니다.

- [Node.js 16.14.0](https://nodejs.org/ko/blog/release/v16.14.0)
- [npm 8.19.2](https://www.npmjs.com/package/npm/v/8.19.2)
- [yarn 1.22.22](https://yarnpkg.com/getting-started/install)

### Installation

```
git clone https://github.com/Sunny-jinn/tgwing.kr.git
cd tgwing.kr
```

### npm

```
npm install
npm run dev
```

### yarn

```
yarn
yarn dev
```

본인의 패키지 매니저에 맞게 실행해주세요.

# Features

### ⭐️ **테크블로그 작성 기능**

- 본인이 학습한 내용을 md 형식으로 자유롭게 작성 가능
- 댓글 및 대댓글 기능으로 의견 공유 가능
- 진행한 프로젝트에서 구현한 기능 소개

### ⭐️ **프로젝트 작성 기능**

- TGThon등 대회에서 진행한 프로젝트나 혼자 진행한 토이 프로젝트 등 소개 가능
- github, notion이나 store 링크를 기제하여 체험할 수 있는 기능
- 프로젝트 진행 과정 및 경험 공유
